# Glossary

The bilingual glossary at `/glossary` defines AI, LLM, prompts, agents, workflows, Runtime states, tools, memory, knowledge/RAG/MCP, tokens, context windows, approval, validation, retries, providers, local storage, and cloud sync.

Each record contains a short definition, example, related lesson, related feature, and availability note. Inline triggers are real buttons, not hover-only tooltips; they support focus, click/tap, and an extended popover. Definitions are structured local content and are also available to Help, Search, and the deterministic Local Assistant.
