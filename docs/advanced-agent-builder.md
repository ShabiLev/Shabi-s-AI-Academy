# Advanced Agent Builder

The Agent Builder adds structured templates, tools and approval policy, memory strategy, retry and error handling, validation rules, completion criteria, local test scenarios, design scoring, version history, and field diffs.

High-risk planned tools require an approval point before save. A saved tool remains a disconnected declaration: Version 1.2.0-beta.1 never represents it as connected and never executes it outside the shared Mock or Dry Run Runtime.
