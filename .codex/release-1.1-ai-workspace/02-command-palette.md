# Command Palette and Shortcuts

Provide a global accessible dialog opened by Ctrl/Cmd+K or Ctrl/Cmd+Shift+P. A typed registry owns bilingual names, aliases, category, risk, availability, reason, and execution descriptor. Navigation, creation, search, language, motion, export, developer mode, recent clearing, shortcut help, and contextual commands use one executor.

The dialog traps focus, restores it, closes with Escape, supports arrows/Enter, exposes disabled reasons, and never confirms or performs destructive actions silently. Command history is local, validated, and bounded. A central shortcut registry also owns search, editor save/run, Escape, and `?` help without intercepting editable fields or unsafe browser shortcuts.

## Acceptance

Every command has deterministic availability and bilingual accessible text.
