# Project Memory

Shabi's AI Academy is a bilingual Hebrew/English, semantic RTL/LTR, local-first React + TypeScript + Vite learning application. It deploys through GitHub Pages (HashRouter) while local development uses BrowserRouter. Supabase authentication and synchronization are optional and remain behind service/provider boundaries. The AOS under `.agent/` coordinates agent-neutral task classification, research, evidence, security, Git, release, memory, and handoff workflows. Vitest, Playwright, axe, Lighthouse, schema checks, and release evidence form the test architecture. Browser storage must never contain secrets; built-in catalogs stay separate from user-owned data. Sources of truth: `AGENTS.md`, `.agent/master.md`, accepted ADRs, `.codex/architecture/`, `.codex/standards/`, the active release specification, and `package.json`.
