# Prompt curation

Every Academy packed prompt has a stable pack ID, bilingual title and description, content language, difficulty, role, task, context, constraints, output format, example, variables, expected output, usage guidance, safety guidance, tags, and source version.

Curators must keep external content inert, remove secrets and personal data, avoid claims of connected tools, include verification instructions, and reject duplicate bodies. Catalog revisions increment source versions while imported user copies remain unchanged.
