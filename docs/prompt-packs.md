# Prompt packs

The beta ships 250 built-in prompts across eleven packs: QA and Testing (40), SQL and Data (30), Jira and Release (25), Product Management (20), Development and Code Review (30), Prompt Engineering (25), Agent Design (25), Professional Communication (20), Learning and Research (15), Automation and Workflows (10), and Security and Risk Review (10).

Built-ins remain immutable. Users may import one prompt, a selection, or a complete pack. Each import receives a new local ID, version 1, source attribution, and editable local ownership. Re-importing creates another copy only after an explicit action.
