# Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| Ctrl/Cmd+K | Open Command Palette |
| Ctrl/Cmd+Shift+P | Open Command Palette |
| Ctrl/Cmd+Shift+F | Open Global Search |
| Ctrl/Cmd+S | Save the current editor |
| Ctrl/Cmd+Enter | Run the current Playground |
| Escape | Close the active overlay or Assistant panel |
| ? | Open shortcut help |

Global shortcuts do not intercept text inputs, textareas, selects, content-editable regions, or unsafe browser combinations. Shortcut help is available from the Command Palette.
