# Beginner and Advanced modes

Beginner Mode is the default for new users. It emphasizes page explanations, four task-oriented Dashboard actions, contextual Help, glossary definitions, and recommended next steps. Technical diagnostics and Developer Mode are progressively disclosed.

Advanced Mode exposes QA, analytics, documentation, release diagnostics, roadmap tools, Runtime detail, and advanced builder controls. It is a presentation preference only: switching modes never grants data access, cloud permissions, or an Admin role.

Change the mode in Settings or Profile. The preference is browser-local and can be changed at any time. Developer Mode remains a separate explicit opt-in available only after Advanced Mode is selected.
