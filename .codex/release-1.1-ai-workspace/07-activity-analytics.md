# Workspace Activity, Notifications, and Analytics

Unify recent open/edit/run/search activity, favorites, and pins for supported entity types. History is bounded, individually clearable, and powers Dashboard continue/recent/favorite/suggestion widgets.

Notifications are browser-local typed records for success, warning, information, action required, release, quality, and storage. Support unread count, read/all-read, delete/clear, and safe internal action links. Never imply server delivery.

Protected `/analytics` shows browser-local counts, categories, recent activity, usage over time, storage estimates, and quality trend with accessible tabular fallbacks. Track only safe metadata events; never prompt/document bodies, chat text outside assistant history, personal data, or secrets. Analytics is bounded, date-filterable, exportable, clearable, and disableable. The storage manager reports per-domain estimates and warns near capacity without silent deletion.

## Acceptance

Clears and resets affect only their declared local domain.
