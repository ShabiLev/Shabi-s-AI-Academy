# Starter agents

The beta includes 32 immutable bilingual starter agents spanning QA, SQL/data, Jira/release, product/communication, development, and AI design. Every template declares its role, goal, inputs, instructions, planned tools, risks, memory, validation, retry policy, approval points, output, completion criteria, quality target, Mock scenario, and sample input.

All conceptual tools are marked `notConnected`. Templates support local preview and import; imported copies receive new local IDs and may be edited without changing the built-in source.
