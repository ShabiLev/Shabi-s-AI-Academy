# Milestones

## Purpose

Define outcome checkpoints without claiming dates that have not been committed.

| Milestone                |     Version | Exit evidence                                                             |
| ------------------------ | ----------: | ------------------------------------------------------------------------- |
| Learning foundation      | 0.1.0–0.3.0 | shell, demo auth, lessons, progress, E2E                                  |
| Creation and quality     | 0.4.0–0.5.0 | Prompt Workshop and QA Center                                             |
| Agent/catalog foundation | 0.6.0–0.6.1 | agent blueprints, How To, curated prompt catalog                          |
| Safe runtime learning    |       0.7.0 | deterministic Mock/Dry Run, catalogs, packs, playgrounds, journeys        |
| Secure integration       |       0.8.0 | server-held credentials, real provider contract and observability         |
| Connected workflows      |       0.9.0 | approved MCP/connectors with permission model                             |
| Public readiness         |       1.0.0 | production auth, privacy/security review, operations and support criteria |

Only 0.7.0 is an active committed Sprint. Later milestones are planned and may change through ADRs and Sprint planning.
