# Complete curriculum

Version 1.2.0-beta.1 includes 45 open bilingual lessons across ten modules: AI foundations, Prompt Engineering, context and verification, models and parameters, agent design, tools, memory, RAG, MCP and automation, and production AI.

Every lesson has objectives, explanatory sections, a practical comparison, common mistakes, exercise, quiz feedback, mini-project, related catalog references, sources, and a content version. Recommended prerequisites never lock a route.

Progress, quiz scores, and drafts use the existing validated browser-local course store. Resetting course progress remains isolated from prompts, agents, projects, runs, and Knowledge Base documents.
