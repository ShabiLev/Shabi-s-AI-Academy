# Agent curation

Starter agents must have complete Hebrew and English names, descriptions, roles, goals, inputs, instructions, risk notes, validation, outputs, completion criteria, quality targets, and samples. Tool declarations use the shared conceptual catalog and always remain `notConnected` in this release.

Curators define bounded retries, explicit stop conditions, and fresh human approvals for risky proposals. Samples contain no credentials or personal data, and catalog content is never executed as code.
