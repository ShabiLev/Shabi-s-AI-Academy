# Onboarding

`/onboarding` is a seven-step, non-blocking local workflow: welcome, main goal, experience, interests, deterministic recommendation, account choice, and finish. Progress is validated and versioned locally. Recommendations map declared goals and interests to existing lessons, packs, starter agents, projects, or tours without AI inference.

New Guest sessions and confirmed registrations may enter onboarding. Existing users with substantive local data receive an invitation rather than forced navigation. Settings can restart onboarding without deleting its prior profile until the user confirms new choices.

## Acceptance

The workflow is skippable, keyboard operable, deterministic, editable later, and never deletes existing local records.
