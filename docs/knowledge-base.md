# Local Knowledge Base

The Knowledge Base stores TXT, Markdown, JSON, CSV, notes, and pasted text in this browser. It supports validated import, title/content/tag/source search, snippets, project links, edit, and delete. The limit is 1 MB per document.

Content is rendered as inert text. It is not uploaded to a provider or synchronized. PDF and DOCX parsing, embeddings, vector search, and RAG are not active.
