# Assistant Sidebar

Add a route-aware assistant to protected workspace routes in collapsed, compact, and expanded modes. It shows an honest local-only notice, screen explanation, current entity summary, relevant prompts/agents/lessons/help, next actions, recent activity, commands, and chat entry.

Desktop layout must not obscure content. Mobile uses a focus-managed drawer/sheet with Escape and browser Back support. Recommendations derive only from current route, indexed workspace data, and registered commands. No provider or connected state may be fabricated.

## Acceptance

Focus restores after close and the local-only notice is always discoverable.
