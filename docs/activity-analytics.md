# Activity and Analytics

Workspace activity records safe metadata for recent items, notifications, favorites, and the `/analytics` dashboard. Analytics shows browser-local counts, categories, activity over time, storage estimates, and quality trend with accessible tables.

Events are bounded, date-filterable, exportable, clearable, and disableable. Prompt bodies, document bodies, Assistant message text, personal identifiers, credentials, and secrets are not analytics fields. The storage manager reports estimates and warnings but never silently deletes user data.
