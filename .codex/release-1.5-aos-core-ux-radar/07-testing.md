# Version 1.5 testing

## Automated scope

Evidence covers schema rejection, hostile prompt-like fields, deduplication, retention, favorites, provider states, Event Bus isolation, Scheduler retry/concurrency, Registry validation, Dashboard removals, Recent Items, navigation, notifications, direction, responsive behavior, accessibility, visuals, and performance.

## Manual scope

Human reviews cover bilingual copy, primary-source summaries, keyboard and screen-reader journeys, responsive layouts, overlay stacking, visual intent, and release content. Their initial state is not run.

## Interpretation

A missing baseline is not a product defect, but a failed mandatory gate still blocks release. Results apply only to the exact tested commit. Visual baselines change only through the separately approved workflow.
