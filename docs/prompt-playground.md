# Prompt Playground

The Prompt Playground assembles system instructions, a user prompt, variables, context, constraints, output format, and examples. It displays the final planned prompt, an approximate token estimate, provider status, warnings, and the resulting Runtime record.

Mock output is simulated. Dry Run is a preview and never invokes a provider. Live remains disabled. Workspaces and run history are browser-local and bounded; saving to My Prompts creates a user-owned editable record.
