# Platform Pages Specification

## Required pages
Public About, Dashboard, Learning Journey, Roadmap, Changelog, QA Center, Release Center, Developer Mode, Documentation Center, How To and Settings.

## About
Public `/about`, explaining purpose, creator, features, privacy, limitations, stack, quality, roadmap, GitHub, version and Vercel URL.

## Dashboard
Real local metrics only: lessons, prompts, agents, runs, projects, KB, QA and milestone.

## Learning Journey
Visual map plus accessible list. All lessons open.

## Roadmap
Typed, honest completed/in-progress/planned versions.

## Changelog
Maintained release history with commit links where known.

## Documentation Center
Indexed user-facing docs; do not expose private engineering-only files unintentionally.

## Release Center
Version, build metadata, quality gates, manual checklist, changelog, production URL and rollback guidance.

## Developer Mode
Protected route explorer, storage inventory, feature flags, mock data, translations, design tokens and quality viewer. No secrets.

## Settings
Language, motion, domain resets, export/import, privacy and Developer Mode access.

## How To
Searchable bilingual help for every major feature.

## Tests
Public/protected routes, honest counts, roadmap, changelog, docs search, release metadata, Developer Mode safety, reset isolation, bilingual, mobile, accessibility and visual.

## Docs
Create page-specific docs for About, Roadmap, Changelog, Release Center, Developer Mode and Documentation Center.
