# UX simplification

- Remove Workspace Overview, Workspace Status, redundant header state, and duplicate Recent Items from Dashboard.
- Keep one canonical, bounded Recent Items history in Profile with filtering, links, empty state, and confirmed clearing.
- Collapse navigation groups by default, reveal the current-route group, and scope persisted preferences per user.
- Notifications close explicitly, with Escape, outside interaction, route navigation, and focus restoration.
- Use semantic logical CSS, visible form borders, responsive layouts, and bilingual RTL/LTR behavior.
