# Prompt library

My Prompts contains user-owned browser-local records. The existing prompts.chat Starter Catalog remains read-only and separately attributed under CC0-1.0. Academy Prompt Packs are another immutable built-in source.

Search, categories, language, favorites, quality and recent ordering help manage local prompts. Imported records preserve source IDs and hashes without storing the complete built-in catalog in localStorage.
