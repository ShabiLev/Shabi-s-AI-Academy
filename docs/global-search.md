# Global Search

Version 1.1.0-beta.1 provides protected route `/search` and shortcut Ctrl/Cmd+Shift+F. A typed local index combines lessons, modules, built-in and personal prompts and agents, packs, projects, knowledge, runs, Help, documentation, roadmap, and changelog records without network access.

Ranking is deterministic and boosts exact phrases, titles, tags, favorites, and recent items. English normalization is accent-insensitive; Hebrew is normalized; practical one-edit typo matching is bounded. Entity, category, language, favorite, recent, and availability filters can be combined. Highlights are React text nodes rather than injected HTML, and history is validated, versioned, and bounded.
